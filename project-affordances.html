WIM: Worlds In Miniature

Description

Prototype implementation of the system in Dynamic Affordances in Embodied Interactive Systems: The Role of Display and Mode of Locomotion Allows users to try and board a train by walking or by joystick, based on an affordance provided by a gate that opens periodically.

Features

  • Infinitely-spawning train
  • Randomly generated intervals for opening and closing the gate
  • Joystick-based and walking navigation

Challenges

The largest challenges of this implementation were designing the user interaction for joystick-based motion, and handling when the user landed on the train.

Controls

  • Dominant-Hand Controller: Joystick Motion

Code

Code from this project can be found in this GitHub repo. This project was implemented with Unity, and used the SteamVR plugin.
