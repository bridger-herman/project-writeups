WIM: Worlds In Miniature

Group Members

  • Bridger Herman
  • Nikki Kyllonen

Description

Prototype implementation of the interaction technique from the paper Virtual Reality on a WIM: Interactive Worlds in Miniature (Richard Stoakley, Matthew J. Conway, Randy Pausch).

Features

  • World in miniature attached to non-dominant hand's controller
  • Selection with dominant hand's controller
  • User avatar display on WIM (allows teleportation)
  • Virtual room with manipulable objects
  • Back face culling for easier WIM viewing

Challenges

The largest challenge we faced was enabling the user to teleport around the scene by picking up their avatar on the WIM. We solved the problem by searching for the SteamVR CameraRig object, and moving it to the proper position with respect to the room (including adjusting for the scale factor between the WIM and the full-size room).

Controls

  • Non-Dominant-Hand Controller: Position the world in miniature
  • Dominant-Hand Controller: Select and manipulate object in the virtual world and in the WIM

Code

Code from this project can be found in this GitHub repo. This project was implemented with Unity, and used the SteamVR plugin.
